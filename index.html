Realtime Data Analysis Platform using AWS

In a team of three, we developed a scalable solution for realtime data analysis using AWS. The project utilized AWS Kinesis to capture streaming data from numerous sources. The data was routed to two destinations: AWS Firehose for continuous ingestion and loading into Elasticsearch and an S3 bucket for storing historical data. Elasticsearch processed the Firehose output, enabling filtered results to be visualized in realtime using Kibana dashboards. For historical analysis, AWS Glue crawlers cataloged the data stored in the S3 bucket, and Amazon Athena provided a SQL-like interface for querying. Finally, Tableau was integrated with Athena to create comprehensive data visualizations for deeper insights.
